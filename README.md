## Overview

> Automated grievance redressal system using NLP for classification.

**AI Driven E Governance Services Hosted on Secure Cloud** is a proprietary web application system engineered by **Karthik Idikuda**. It leverages TypeScript, React, Next.js, TailwindCSS for its core functionality.
